Instalador do Joomla



     Dica: Todos os campos com -backup estão no configuration.php do site ou no arquivo zip do backup.


* Todos os campos são obrigatórios.

Ajuda sobre o Instalador do Joomla

Este instalador requer dois arquivos, o .zip contendo todos os arquivos do site em Joomla e o .sql contendo todo o banco de dados. Eu usei o componente SimpleBackup para criar estes arquivos mas pode ser qualquer outro, desde que crie estes dois arquivos.

Instalação Local

Criação do Banco de dados - desnecessária se usar o root


Instalação num Servidor Remoto

Se não lembra os dados do banco então abra o configuration.php do site e anote host, user, password, db, log_path, tmp_path

É recomendado que este software esteja no mesmo servidor onde o joomla será instalado e idealmente proteja seu diretório para que seja usado somente por você ou o remova após a instalação.

Obrigatório:

Criar o banco de dados, criar o usuário e dar privilégios ao usuário sobre o banco de dados

Renomeie o arquivo.sql do backup para site.sql

E renomeie o arquivo.zip para site.zip

Então os copie para a pasta do joomla_install

Pasta - é onde será instalado o site atual. Caso não exista será criada.
