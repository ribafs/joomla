Não tente burlar o algoritmo no Google
Conheça o Black Hat, prática que contraria as diretrizes pelo bom rankeamento nas páginas
Você sabe o que é 'Black Hat'? Trata-se de um conjunto de práticas cujo objetivo é melhorar o ranqueamento de um site no Google, entretanto, de forma contrária às diretrizes da empresa. Na maioria das vezes, isso prejudica também as boas práticas de usabilidade na web e experiência do usuário.

Essas técnicas são desenvolvidas para tentar burlar e enganar alguns fatores do algoritmo do Google, influenciando diretamente nas páginas de resultados de busca. Pode-se perceber então como as técnicas de Black Hat chamam atenção para quem está desenvolvendo uma campanha de SEO.

Existem ainda outras duas técnicas de otimização, se quiser saber mais, veja sobre Gray Hat e White Hat, além da Black Hat aqui.

O que acontece? 

O algoritmo do Google também busca por sites que estejam fazendo 'Black Hat' e, quando consegue identificá-los, os pune.  Veja como a punição do Google é famosa por conseguir quase que empurrar um site para o ostracismo, já que a maior parte dos acessos do site vem pelos resultados orgânicos da ferramenta de busca.

Ou seja, não vale a pena tentar burlar o algoritmo do Google, pois pode ser fatal para você, e, mais cedo ou mais tarde, o Google vai conseguir te achar.


Quais são as técnicas? 

  • Compra de links
Esta é uma das formas de link building mais comuns no Black Hat, que tem intenção de manipular os fatores de ranqueamento baseados na análise dos links que apontam para o site. Esses links são posicionados dentro de textos do site, geralmente com o texto âncora envolvendo palavras-chave específicas.

Embora o Google esteja se desenvolvendo para identificar esses casos, ainda é relativamente difícil saber quais links são pagos ou autênticos.

  • Keyword stuffing
No início das ferramentas de busca, um dos fatores de ranqueamento era a quantidade de repetições da palavra-chave no conteúdo de uma página específica. Com isso, os responsáveis pelos sites repetiam a palavra-chave indiscriminadamente, muitas vezes fazendo o texto ficar sem sentido, prejudicando a experiência do usuário.  Há tempos o Google consegue identificar isso com facilidade e essa prática quase não é mais utilizada.

  • Cloaking
Este é o termo em inglês para especificar o conteúdo que se 'esconde'. Imagine uma capa que torna a pessoa invisível quando algum inimigo está chegando perto. É algo parecido com isso que é feito no cloaking. Uma mesma página (URL) é exibida para o usuário e outra para o robô do Google.

Sempre que este robô acessa o site, a página exibida é totalmente otimizada para o buscador. Isso quer dizer que a página será bem avaliada, mas não pelo conteúdo verdadeiro, e que será  exibido para o usuário, mas por outra, que é feita somente pensada no robô. É fácil entender porque é considerada Black Hat.

  • Esconder conteúdo
Esta técnica tenta enganar os mecanismos de busca ao exibir uma maior quantidade de conteúdo otimizado para o robô de análise, embora ele não apareça para o usuário, para não prejudicar a sua experiência. Imagine uma página com textos por todas as partes, deixando o site muito poluído e com difícil leitura.

Para não deixar o site poluído, os responsáveis pelo site colorem o texto com a mesma cor do fundo, tornando-o “invisível” para os usuários.
